# Wireframe City Shooter

スター・フォックスやスペースハリアー系のフリーフライトを意識したワイヤーフレーム3Dシューティング。ハンガー発進（Stage 0）から都市空域（Stage 1）、宇宙遷移（Stage 2）、巨大戦艦～最終ボス戦までを一枚HTMLで遊べるように仕上げています。

## 現行ステータス（2025-11-26時点）
- 主要ステージ/ボス/敵挙動・BGM・自機武装は完了済み（plan.md Progress: 99%）。
- ローリング無敵＆敵弾反射、シールド/HP/ストック、誘導ミサイル、TypedArray経由の高速描画など実装済み。
- Vision-AI向けの **AI Status Panel**（黒地/白文字/モノスペース）と説明ラベルを追加。Stage 0 のローンチシーケンス開始から約12秒間、`H/E/S/EV` の意味を字幕として表示しており、OCRで即座に読み解けます。

## 操作方法
- `Arrow Keys / WASD` : 前後左右移動
- `Space` : メインショット
- `KeyF` : 単発ミサイル発射（同時発射4本まで）
- `ArrowRight → ArrowLeft → ArrowRight` をテンポ良く入力 : ローリング無敵+反射
- サウンドテストは右上の `Sound Test` ボタンで開閉

## ビルド / 実行
1. 依存パッケージなし。`index.html` をモダンブラウザで開くだけで動作。
2. オーディオ再生は初回ユーザー操作後に有効化される（ブラウザのAutoplay制限対策）。
3. Vision-AI用途で `#aiPanel` の150x100キャンバスを常時描画。ピクセル等倍＆white-on-black固定でOCRが乱れないようチューニング済み。
